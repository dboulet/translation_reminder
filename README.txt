This module adds options at the end of node forms to proceed to creating translations after saving the node. This is useful for reminding content creators to translate the content that they create or edit.

INSTALLATION
============
- Enable the module on the Modules page
The translation reminder will not appear at the bottom of node forms for all node types that have translation enabled.


DEPENDENCIES
============
- Content translation (Core module)
